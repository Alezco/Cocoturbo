# Cocoturbo
PHP Project

## Setup

Clone the repository then
```sh
$ composer install
```
Create cocoturbo database then
```sh
$ php artisan migrate
$ php artisan db:seed
$ php artisan serve
```
Enjoy on localhost:8000
